# yesterday-weather 개선 계획

## 현재 상태

- 기본 기능 완성: 어제 날씨 조회, 도시 검색, PWA 지원
- Bun + Hono + Alpine.js 스택
- Open-Meteo Archive API (무료, 키 불필요)
- Fly.io (nrt) 배포 — 가동 중 (https://yesterday-weather.fly.dev/)

## 철학

> "날씨를 기능적 데이터가 아닌 하루의 분위기와 기억을 구성하는 요소로 다룸"

입력 없음, 클릭 없음, 스크롤 없음 — 단일 화면.
*사업은 아니지만, 남길 가치는 있는 프로젝트.*

## 현재 기능

- 위치 기반 또는 도시 선택으로 어제 날씨 표시
- 80개+ 인기 도시 목록 (아시아, 유럽, 미주 등)
- Nominatim으로 도시 검색
- 온도 단위 토글 (C/F, 국가 기반 자동 선택)
- localStorage로 마지막 도시 기억
- Pull to refresh
- PWA 설치 지원

## 개선 아이디어

### 1. 날씨 타입 시각화
현재 weatherType을 API에서 받지만 UI에 미사용.
- 배경색/분위기 변화 (clear=밝음, rain=어둡고 차분함)
- 날씨 타입별 이미지 추가 (README에 언급됨: `public/images/`)
- 또는 CSS 그라데이션으로 미니멀하게 표현

### 2. 날씨 아이콘 또는 이모지
- 온도 옆에 작은 아이콘/이모지로 날씨 상태 표시
- 예: "5°C ☁️" 또는 별도 심볼

### 3. 다중 날짜 (과거 기록)
- 어제뿐 아니라 최근 7일 날씨 확인
- 스와이프로 날짜 이동

### 4. 공유 기능
- "어제 서울은 5°C, 흐림" 같은 텍스트 복사/공유

### 5. 위젯
- iOS/Android 위젯 지원 (PWA 한계 있음)
